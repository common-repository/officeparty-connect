Use Officeparty Connect to integrate your WooCommerce store with your Officeparty community.

== Description ==

Officeparty Connect integrates WooCommerce stores with the Officeparty platform. Simply activate the plugin and configure the store on Officeparty to get started.

### New to Officeparty?

Check us out at officeparty.co.

### Support

Reach out to us at info [at] officeparty [dot] co.

== Installation ==

### Automatic installation

1. Click the `Add New` button in the 'Plugins' menu in WordPress
1. Search `Officeparty Connect` 
1. Click `Install`
1. Click `Activate`

### Manual installation

1. Download `officeparty-connect.zip`
1. Upload the plugin file in the 'Plugins' menu in WordPress
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How do I get started using Officeparty? =

You can sign up for an account at officeparty.co

= Do I need to configure anything? =

Not on WordPress. All configuration should be Officeparty through the E-Commerce tab. Set up should be self-explanatory.

= Who should I contact for support? =

Send an email to `info [at] officeparty [dot] co`, and we will get back to you ASAP.
